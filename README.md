# Aquaman Services — Mobile Detailing Memphis

Premium mobile detailing website.

## Deploy to Vercel

### Option 1 — Drag & Drop (easiest)
1. Unzip this folder
2. Go to https://vercel.com/new
3. Drag the entire unzipped folder onto the upload zone
4. Click **Deploy**
5. Your site is live at `your-project.vercel.app`

### Option 2 — Vercel CLI
```bash
npm i -g vercel
vercel
```

## Local Development
Just open `index.html` in any browser — no build step needed.

## Files
- `index.html` — entry point
- `components.jsx`, `sections.jsx`, `booking.jsx`, `app.jsx` — React components
- `videos/` — all video assets (hero, enhancement reels, testimonial reels, outro)
- `images/` — logo and feature imagery
